# CTA Aneurysm Detection using YOLOv8

This repository contains a complete pipeline to prepare CTA DICOM data, clean metadata, generate YOLO-formatted datasets, train a YOLOv8 object-detection model, and perform inference on new CTA scans.

# Project Overview

The goal of this project is to automatically detect intracranial aneurysms from CTA (Computed Tomography Angiography) slices using YOLOv8.
